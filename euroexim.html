Automated KYC modeule for Bank App

Feb, 2018 - June, 2018

KYC has now become one of the basic processes that every user of every bank needs to go through. Even after being so commonplace it still requires considerable human involvement. At Euro Exim, a step was taken to automate this process. The users are required to upload an image of an official document (Passport, Driver License etc) the KYC features captures a set of images from the webcam of the system or mobile phone's front camera, from the pictures, clicked the algorithm detects the front face pose, crops the face and compares it with the face cropped from the official document and gives the match percentage. The bank then decides the confidence level at which the algorithm says the people are matched or unmatched.

Knowledge Gained

  1. Implementing transfer learning over pretrained Deep Learing Models.
  2. One Shot learning for reducing a considerable number of epocs required by the Deep Learning model to train on unseen data to one
  3. Tackelling the problem of training the neural network from just two images
  4. Implementing Siamese neural net for handling image pair parallely
  5. Using Deep Learning Models not as a classifier but as a feature extractor.
  6. Using different distance methods and thresholdings techniques.
