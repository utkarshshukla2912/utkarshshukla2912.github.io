Platform For Development, Compilation And Submission of Codes for Assessment

CodeRanks is an online portal where students of Vellore Institute of Technology can improve their problem solving skills (which helps them perform well in competitive exams), programming skills by taking various practice tests. Students and Authorities can access various actionable analytics(reports like strength and weakness) which helps them to improve their skills.

Features

  1. Quick compilation of codes for Python, C, C++, JAVA languages.
  2. Frequent assesment for scoring of different users.
  3. Analytics showing strengths, weakness, frequency of coding and pending assignment.
  4. Dashboard intigration for authority to view different users of the portal and assess the users based on the submission scores.

Knowledge Gained

  1. Using distributed deployment of webapps
  2. Intigration of Python Django Framework with Java Based compiler.
  3. Database Creation and management, making nested queries with Transaction Control Language
  4. Developing interactive visualisation
  5. Handling MYSQL ingestions for making the platform tolerant against attack over the Database Management.
